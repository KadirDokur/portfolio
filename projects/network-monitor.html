> network-monitor

A Python-based network monitoring tool designed to observe real-time traffic, analyze connections, and provide insights into network activity. Built for learning and practical network analysis.

> Features

  • Real-time traffic monitoring
  • Active connection tracking
  • Basic packet inspection
  • Lightweight CLI interface

> Technologies

  • Python
  • Socket / Scapy
  • Networking fundamentals

> Purpose

Developed to understand how network communication works and how traffic can be monitored in a controlled environment.
